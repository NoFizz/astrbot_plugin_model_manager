# 模型管理器 (Model Manager)

**统一管理 AstrBot 所有插件的 LLM 模型配置**

一个页面集中查看和修改所有插件的模型分配，无需修改任何插件代码。

## 功能

- **扫描**：自动扫描所有已安装插件的 `_conf_schema.json`，识别 `select_provider` 类型的配置项
- **展示**：在一个页面集中列出所有插件的模型选择配置及其当前值
- **修改**：直接在下拉框中切换模型，支持批量保存
- **快速替换**：一键将所有使用某款模型的配置项替换为新模型
- **全部设置**：一键将所有配置项设置为同一款模型
- **插件排序**：自定义插件显示顺序，支持上下移动并记住排序
- **中英文切换**：支持中文/英文界面，语言设置自动保存
- **零侵入**：无需修改任何其他插件代码，通过读写配置文件实现

## 安装

### 通过 AstrBot 插件市场

在 WebUI 插件市场搜索"模型管理器"直接安装。

### 手动安装

```bash
cd AstrBot/data/plugins
git clone https://github.com/NoFizz/astrbot_plugin_model_manager.git
```

然后在 WebUI 插件管理页面刷新并启用即可。

## 使用方法

1. 在插件详情页打开 **Model Manager** 页面
2. 页面自动扫描并列出所有插件的模型配置项
3. 在每个下拉框中选择想要分配的模型
4. 点击 **保存** 批量保存修改

### 快速替换

点击 **快速替换** 按钮，选择当前模型和新模型，一键替换所有使用当前模型的配置项。

### 全部设置

点击 **全部设置** 按钮，选择目标模型，一键将所有配置项设置为该模型。

### 插件排序

点击插件卡片左侧的上下箭头按钮，可调整插件显示顺序，排序会自动保存。

### 语言切换

点击左上角 🌐 按钮，可在中文/英文之间切换，语言设置会自动保存。
